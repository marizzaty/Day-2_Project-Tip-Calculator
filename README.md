# Tip Calculator

Calculator to calculate tip by letting user inputting their desire percent.

## Installation

Copy this repository link and clone it.
```
$ git clone https://github.com/marizzaty/Day-2_Project-Tip-Calculator.git
```
## Usage
User have to input a few times.

```
Welcome to Tip 10/12/15/20 Calculator.
The total bill: $
```

Here user can input any number. No need for currency sign.

```
How much tip you want to give? 10/12/15/20? 
```

Oppose to the questions, user can input any percentage they want. The number only for references.

```
How many people will split for the bill?
```

Lastly, it will ask how many people will split the bill.
